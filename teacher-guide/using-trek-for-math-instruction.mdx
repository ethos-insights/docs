---
title: "Using Trek for Math Instruction"
description: "How to use Trek's tools to support math literacy and problem-solving."
---

Although Trek is primarily a literacy platform, its tools support math instruction through math-focused reading passages, word problem scaffolds, and writing-to-explain tasks.

## Math nonfiction passages

Trek's library includes nonfiction articles on math topics — data literacy, real-world applications of algebra, financial math, and more. These can be used to:

- Build background knowledge before a new unit
- Practice reading informational text in a math context
- Connect math concepts to real-world scenarios

Search the library using the **Subject: Math** filter.

## Writing to explain math thinking

Trek's writing tools support mathematical explanation and argumentation:

- **Constructed response prompts** — Ask students to explain their reasoning or justify a solution in writing.
- **Evidence-based writing** — Students can cite data from a passage to support a mathematical claim.
- **Worked example annotations** — Attach an image of a worked problem and ask students to annotate or explain each step.

## Vocabulary support for math

Use Trek's **word bank** feature to front-load key math vocabulary. Students can reference the word bank while writing explanations, reducing the cognitive load of recalling terms and letting them focus on reasoning.

## Recommended assignment types for math

| Goal | Recommended assignment type |
| --- | --- |
| Build math vocabulary | Vocabulary assignment with word bank |
| Practice explaining reasoning | Short constructed response |
| Connect math to real world | Reading \+ writing assignment with a math nonfiction passage |
| End-of-unit reflection | Extended writing prompt tied to a driving question |

<Tip>
  Math teachers who pair Trek with problem sets report stronger student explanations and better performance on written response questions on standardized tests.
</Tip>
